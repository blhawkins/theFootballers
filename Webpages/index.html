The Footballers

Welcome to our project!

We are Benji Hawkins, David Seger, and Rachidi Ndongala!

For this project, we are collecting and visualizing information about the most valuable association football (soccer) players in the world. To begin, we used Python's Beautiful Soup and Splinter libraries to scrape information from the SoFIFA.com website. Then, we used PyMongo and a Flask server to upload the scraped data to a Mongo database. The data was subsequently used to create the Javascript, Plotly.js, and Leaflet.js visualizations shown on this website.

Please take a look around our site! It can be navigated by using the navigation bar present at the top of the page.

If you have any questions or would like to see the source code for this project, please consult our GitHub repository!
